## Content Differentiation Requirements

### Never Create Generic Content About
- Basic woodworking advice available in countless DIY websites and beginner guides
- One-size-fits-all approaches ignoring skill level differences and project complexity
- Oversimplified technique instruction that doesn't address safety requirements and quality standards
- Generic tool recommendations without comprehensive evaluation of quality and application

### Always Create Definitive Resources For
- Evidence-based woodworking instruction with current safety protocols and traditional techniques
- Skill-appropriate approaches that address different experience levels and learning progressions
- Comprehensive technique development that ensures safety while promoting craftsmanship excellence
- Quality-focused instruction that honors traditional methods while embracing modern improvements
- Community-connected guidance that promotes craft tradition continuation and skill sharing

**Each article must pass these tests:**
1. **Craft Standards**: Would master woodworkers and craft instructors endorse this guidance?

5. **Traditional Respect**: Does this honor craft traditions while supporting individual creative development?

**The Ultimate Test**: If someone could only read one resource about this woodworking and craftsmanship topic, would this guide them to develop safe, quality woodworking skills that honor traditional craftsmanship while building practical competency and creative expression through proper technique and respectful material stewardship?
