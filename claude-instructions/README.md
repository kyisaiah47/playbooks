## 🎯 Usage Instructions

**For New Claude Sessions:**
1. Always read `claude-template-instructions.md` first for template work
2. Reference `blog-generation-guidelines.md` before creating blog content
3. Use `resource-viewer-styling.md` for UI/styling reference
4. Follow the established patterns and quality standards

**File Updates:**
When adding new guidelines or instructions:
1. Place them in this `claude-instructions/` directory
2. Update this README.md to include the new file
3. Use clear, descriptive filenames
4. Include validation checklists where applicable
